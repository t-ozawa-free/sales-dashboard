# sales-dashboard

業務データ分析ダッシュボード

## 技術スタック
- Python 3.12
- pandas
- Streamlit
- plotly


## 要件定義
## 1. 概要
CSVやExcelでまとめられている業務・売上データを読み込み、自動分析・可視化を行うダッシュボード

## 2. 主要機能

### 2.1 データ読込
- CSVファイルのアップロード機能
- データプレビュー表示

### 2.2 前処理(自動実行)
- 欠損値の補完(平均値/中央値/前後の値)
- 異常値の検出と対応(統計的手法)
- データ型の自動変換(日付、数値など)

### 2.3 分析・可視化機能
**必須機能(Week 1-2で実装)**
- 売上推移グラフ(時系列: 日別/月別)
- 前月比・前年比の表示
- 基本統計量(合計、平均、最大最小)

**追加機能(Week 3で実装・時間次第)**
- 商品/カテゴリ別分析(円グラフ・棒グラフ)
- 異常値の可視化(外れ値の強調表示)

## 3. 技術スタック
- Python 3.12
- pandas: データ分析
- Streamlit: UI・可視化
- plotly: グラフ作成
- Faker: サンプルデータ生成

## 4. サンプルデータ仕様
**売上データ(Fakerで生成)**
- 日付(date): 2024年1月1日〜12月31日
- 商品名(product_name): 9種類(カテゴリごとに3種類)
- カテゴリ(category): 3種類(電子機器、文房具、書籍)
- 単価(unit_price): 150円〜80,000円
- 数量(quantity): 1〜10個
- 売上金額(sales_amount): unit_price × quantity
- 顧客ID(customer_id): customer_001〜customer_100

**データ件数**: 約1,461件(1日あたり3〜5件)

**データ品質(意図的に混入)**
- 欠損値: 5%程度
- 異常値: 2-3件(極端に高い/低い売上)
